Overview

The goal was to build a lightweight operating system for creative teams that live in async. The product needed to feel calm, to work offline, and to be shippable in a single quarter.

I designed a modular layout system, set up the data architecture, and built the core dashboard experience with an emphasis on speed and clarity.
